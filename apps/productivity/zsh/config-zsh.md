## Theme

- https://zshthem.es/all/
- https://github.com/ohmyzsh/ohmyzsh/wiki/Themes

Abra o arquivo `/home/.zshrc` e edite a linha abaixo:
``` sh
ZSH_THEME="clean"
```
  
### Plugins

- https://github.com/ohmyzsh/ohmyzsh/wiki/Plugins

Abra o arquivo `/home/.zshrc` e edite a linha abaixo:

``` sh
plugins=(
    git
    node
    vscode
    npm
    ng
)
```
